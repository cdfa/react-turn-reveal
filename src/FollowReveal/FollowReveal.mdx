# FollowReveal

A `<FollowReveal>` is a like a `<TurnReveal>` that follows the mouse's movements in and out of the component.

import { Props, Playground } from "docz";
import FollowReveal from "./FollowReveal";
import { Front, PlaceHolderImage, Masonry } from "docs/PlayGroundUtils";

<Props of={FollowReveal} />

## Basic Usage

<Playground style={{ display: "flex", justifyContent: "center" }}>
  {() => {
    const perspective = 400;
    return (
      <div style={{ position: "relative", perspective: perspective + "px" }}>
        <FollowReveal perspective={perspective}>
          <Front>More info</Front>
        </FollowReveal>
        <PlaceHolderImage />
      </div>
    );
  }}
</Playground>

## Masonry Layout

Looks even cooler in a Masonry layout.

<Playground>
  {() => {
    const perspective = 400;
    return (
      <Masonry>
        {["a", "b", "c", "d", "e", "f", "g"].map(area => (
          <div
            key={area}
            style={{
              gridArea: area,
              position: "relative",
              perspective: perspective + "px"
            }}
          >
            <FollowReveal perspective={perspective}>
              <Front>More info</Front>
            </FollowReveal>
            <div
              style={{
                background: "#e2e2e2",
                width: "100%",
                height: "100%"
              }}
            />
          </div>
        ))}
      </Masonry>
    );
  }}
</Playground>
